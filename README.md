# Test Reports

测试报告托管仓库，通过 GitHub Pages 自动发布。

## 访问地址

https://xichencx.github.io/test-reports/

## 使用方式

1. 将 HTML 测试报告放到仓库任意目录下
2. 推送到 `main` 分支
3. GitHub Actions 自动部署，首页自动生成报告索引

## 示例

```
reports/
  unit-test.html      → https://xichencx.github.io/test-reports/reports/unit-test.html
  2026/04/e2e.html    → https://xichencx.github.io/test-reports/2026/04/e2e.html
```
